# All Risk No Reward

All Risk No Reward is a challenge application currently under development. It gives players randomized, level-based challenges while keeping every activity legal, consent-respecting, and safety-filtered.

## Getting Started

Project setup and usage instructions will be added as development progresses.

## Challenge Library

The application-ready dataset is in [`data/challenges`](data/challenges/README.md). It currently contains **375 original same-day challenges** split evenly across five levels:

- Easy
- Medium
- Hard
- Extreme
- Nightmare

Each level contains 75 challenges. The directory includes a manifest, JSON Schema, stable challenge IDs, research notes, and a required safety policy. Higher levels increase effort, time, planning, travel, or social courage—not danger or humiliation.
